# TechNest Landing Page

## 📌 Project Overview

TechNest Landing Page is a responsive and modern website created using HTML, CSS, and JavaScript. This project was developed as part of an internship task to demonstrate front-end web development skills and responsive design principles.

The website provides an engaging user interface with a clean layout, smooth navigation, and mobile-friendly design to ensure a seamless experience across different screen sizes.

---

## 🚀 Features

- Responsive Design (Mobile, Tablet, Desktop)
- Modern and Clean User Interface
- Interactive Navigation Menu
- Smooth Scrolling Effects
- Hero Section with Call-to-Action
- Services/Features Section
- Contact Section
- Optimized Layout for Different Devices
- Cross-Browser Compatibility

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
